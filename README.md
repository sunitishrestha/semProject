Node.js + PostgreSQL Starter

Simple Node.js starter with PostgreSQL, Sequelize, JWT authentication, and Docker.

Stack
	•	Node.js (JavaScript)
	•	PostgreSQL
	•	Sequelize ORM
	•	JWT Auth (access + refresh tokens)
	•	BullMQ + Redis
	•	Docker & Docker Compose

Features
	•	JWT authentication
	•	Background jobs
	•	Database migrations
	•	Input validation
	•	Dockerized setup

Run with Docker

make dev    # Development
make prod   # Production
